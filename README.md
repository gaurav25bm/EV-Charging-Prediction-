# ⚡ Smart EV Charging Assistant
A Machine Learning web application that predicts the optimal charger type, energy requirements, and charging costs for Electric Vehicles.

### 🚀 Features
* **Charger Classification:** Uses a trained model to suggest Level 1, Level 2, or DC Fast Charging.
* **Energy Regression:** Estimates total kWh needed based on battery capacity and State of Charge (SOC).
* **Cost Calculation:** Provides real-time cost estimates based on user inputs.

### 🛠️ Built With
* **Python** (Pandas, NumPy, Scikit-Learn)
* **Streamlit** (Frontend UI)
* **GitHub** (Version Control & Deployment)
